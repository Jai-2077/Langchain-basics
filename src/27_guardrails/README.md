# Guardrails — Safety and Content Filtering

# 🛡️ Guardrails

## What are Guardrails?
**Guardrails** = safety checks that prevent LLMs from producing harmful output.

## Types of Guardrails
- **Input guardrails** — filter harmful user input
- **Output guardrails** — filter harmful LLM output
- **Topic guardrails** — restrict LLM to specific topics
- **PII guardrails** — remove sensitive data
- **Hallucination detection** — fact-check LLM answers
