# Embeddings — Turning Text into Vectors

# 🔢 Embeddings

## What are Embeddings?
An **embedding** converts text into a list of numbers (a vector).

```
"Hello, world!" → [0.23, -0.45, 0.67, ..., 0.12]  (1536 numbers)
```

## Why Vectors?
Computers can't understand words. But they can:
- Measure **distance** between vectors
- Find **similar** vectors
- Perform **arithmetic** on vectors

## The Magic: Semantic Similarity
```
"dog" → [0.2, 0.8, ...]
"puppy" → [0.19, 0.81, ...]  ← similar!
"quantum" → [0.9, 0.1, ...]  ← very different
```
