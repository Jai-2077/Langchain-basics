# 🕸️ LangGraph Basics

## What is LangGraph?
LangGraph is a library for building **stateful, cyclic** AI workflows.

## Why LangGraph?
LCEL chains: A → B → C (linear, no loops)
LangGraph: Can loop back, branch, and maintain state

## Key Concepts
| Concept | Description |
|---------|-------------|
| Graph | The workflow structure |
| Node | A processing step (Python function) |
| Edge | Connection between nodes |
| State | Shared data passed between nodes |
| Conditional Edge | Branch based on s
