# 🖼️ Multi-Modal LLMs

## What is Multi-Modal?
Multi-modal LLMs can process **multiple types of input**:
- Text
- Images
- Audio (some models)
- Video (some models)

## Supported Models
- GPT-4o: text + images
- Claude 3.x: text + images
- Gemini: text + images + audio + video

## Use Cases
- Image analysis
- Document OCR
- Chart/graph reading
- Visual QA
- Medical imaging analysis
