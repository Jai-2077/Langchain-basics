# 💾 Chat History

## What is Chat History?
**Chat history** = stored record of past messages in a conversation.

## Storage Options
| Storage | Package | Best For |
|---------|---------|---------|
| In-Memory | langchain-core | Development/testing |
| Redis | langchain-community | Production, fast |
| PostgreSQL | langchain-community | Production, relational |
| MongoDB | langchain-mongodb | Production, document |
| DynamoDB | langchain-aws | AWS deployments |
| File System | langchain-community
