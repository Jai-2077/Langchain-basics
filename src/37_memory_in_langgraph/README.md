# 🧠 Memory in LangGraph

## Two Types of Memory
1. **Short-term (in-thread)**: conversation history within one session
2. **Long-term (cross-thread)**: facts persisted across all sessions

## Short-Term Memory
Managed by checkpointers — conversation history per thread_id.

## Long-Term Memory
Stored separately — user preferences, facts, summaries that persist across all conversations.
