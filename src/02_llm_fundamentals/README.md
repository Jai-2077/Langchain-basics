# 🧠 LLM Fundamentals

## What is an LLM?
A **Large Language Model (LLM)** is a neural network trained on massive amounts of text to predict the next token.

## Mental Model: The Autocomplete Genius
Think of an LLM as the world's most sophisticated autocomplete:
- Your phone's autocomplete suggests the next word based on recent context
- An LLM suggests the next **token** based on ALL the text ever written
