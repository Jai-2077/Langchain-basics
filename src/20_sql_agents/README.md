# 🗃️ SQL Agents

## What is a SQL Agent?
A SQL agent converts **natural language questions** into **SQL queries** and returns results.

```
User: "How many users signed up last month?"
Agent: SELECT COUNT(*) FROM users WHERE created_at >= '2024-01-01'
Result: 1,247 users
```

## Use Cases
- Business intelligence without SQL knowledge
- Automated reporting
- Data exploration chatbots
- Executive dashboards
