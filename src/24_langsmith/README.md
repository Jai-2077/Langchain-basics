# LangSmith — Observability and Debugging

# 🔭 LangSmith

## What is LangSmith?
**LangSmith** is Anthropic's observability platform for LangChain applications.

## What it Does
- **Traces** every LLM call, tool call, chain step
- Shows exact prompts sent and responses received
- Tracks latency and token usage
- Enables prompt testing and evaluation
- Production monitoring

## Why Observability?
Without observability, debugging AI apps is like:
- Flying blind
- No logs, no metrics
- Can't tell why the LLM gave a bad answer
