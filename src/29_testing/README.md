# 🧪 Testing LangChain Applications

## Why Testing is Hard
LLMs are non-deterministic. Testing them requires:
- Mocking LLM responses
- Evaluating output quality (not just equality)
- Testing at multiple levels

## Testing Levels
1. **Unit tests** — test individual components with mocked LLMs
2. **Integration tests** — test chains end-to-end
3. **Evaluation** — assess output quality with LLM-as-judge
