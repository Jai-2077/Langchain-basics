# 🗂️ Subgraphs

## What are Subgraphs?
A **subgraph** is a complete LangGraph graph used as a **node** inside another graph.

## Why Subgraphs?
- **Modularity**: break complex graphs into manageable pieces
- **Reuse**: same subgraph in multiple parent graphs
- **Encapsulation**: hide implementation details
- **Team collaboration**: different teams own different subgraphs
