## Why This Module Exists
Before touching LangChain, you need **rock-solid Python fundamentals**.
LangChain uses advanced Python patterns extensively:
- **Type hints** → every LangChain object is typed
- **Dataclasses / Pydantic** → LangChain's entire data model
- **Generators** → streaming responses
- **Async/await** → production performance
- **Decorators** → `@tool`, `@chain`
- **Context managers** → tracing, callbacks
